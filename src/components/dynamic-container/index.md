## API

| ---------------- | ---------------------- | ------------------ | ------ |
| CustomContainer  | 自定义容器             | ComponentType<any> |        |

| isCard           | 是否为 SCard 容器      | boolean            | true   |
| hasBottomPadding | SCard 是否包含底边边距 | boolean            | true   |
